# 고도화 아이디어

이미지 OCR 앱을 발전시킬 아이디어 모음. 체감 효과와 구현 난이도를 함께 표기했습니다.
(효과 ★ 많을수록 사용자 체감 큼 · 난이도 = 구현 비용)

## 🚀 우선순위 높음 (효과 크고 저렴)

| 아이디어 | 설명 | 효과 | 난이도 |
|---|---|---|---|
| **영역 선택 OCR** | 이미지에서 드래그로 사각형 영역을 지정해 그 부분만 인식 | ★★★ | 중 |
| **결과 좌표 하이라이트** | 인식된 단어를 이미지 위에 박스로 표시, 텍스트↔이미지 연동 | ★★★ | 중 |
| **언어 선택 UI** | kor/eng 외 언어를 체크박스로 켜고 끄기(tessdata 자동 감지) | ★★ | 하 |
| **전처리 토글** | 흑백화·대비강화·이진화·확대 후 인식(저해상/사진 정확도↑) | ★★★ | 중 |
| **여러 파일 일괄 처리** | 폴더/다중 파일 드롭 → 순차 OCR → 결과 일괄 저장 | ★★ | 하 |
| **설정 저장** | 마지막 엔진·언어·창 크기·전처리 옵션 기억 | ★★ | 하 |
| **드롭 하이라이트** | 드래그 중 창 테두리 강조 등 시각 피드백 | ★ | 하 |

## 🧠 인식 품질

- **Tesseract PSM/OEM 튜닝**: 페이지 분할 모드(PSM) 선택(한 줄/단락/자동), 문서 유형별 프리셋.
- **자동 회전/기울기 보정(deskew)**: 비뚤어진 스캔 자동 정렬.
- **tessdata_best ↔ fast 전환**: 속도/정확도 프로필 선택.
- **제3 엔진 옵션**: PaddleOCR(한글 강함) 또는 클라우드 OCR(Azure/Google) 플러그인.
- **후처리 사전**: 자주 틀리는 문자쌍 교정, 맞춤법/사전 기반 보정.

## 📤 입출력 & 연동

- **PDF 입력**: 페이지별 렌더 후 OCR(다중 페이지 문서).
- **검색 가능한 PDF 출력**: 원본 이미지 + 투명 텍스트 레이어.
- **구조화 출력**: JSON/TSV(단어별 좌표·신뢰도), 마크다운 표 인식.
- **클립보드 자동 감시**: 캡처하면 자동으로 OCR(단축키/트레이).
- **전역 단축키 캡처**: 화면 영역 캡처 → 즉시 OCR(스니핑 툴 대체).
- **번역 연동**: 추출 후 원클릭 번역(DeepL/파파고).
- **결과 히스토리**: 최근 OCR 목록·재열람·검색.

## 🎨 UX / 완성도

- **다크/라이트 테마 전환**(DevExpress 스킨).
- **이미지 확대/이동(팬·줌)**, 회전 버튼.
- **진행률 표시**: 대용량/일괄 처리 시 프로그레스 바.
- **줄/단락 단위 복사** 버튼, 결과 편집 후 저장.
- **접근성**: 키보드 내비게이션, 스크린리더 라벨, 고대비.
- **UI 다국어화(i18n)**: 한/영 UI 전환.

## 🔧 배포 / 운영

- **exe 용량 다이어트**: 현재 self-contained 173MB → framework-dependent(런타임 별도) 또는 트리밍으로 축소.
- **코드 서명**: SmartScreen 경고 제거(인증서 필요).
- **MSIX/Installer**: 시작 메뉴 등록·자동 업데이트 대안.
- **업데이트 개선**: 백그라운드 다운로드, 델타 업데이트, 서명 검증, "나중에" 옵션, 채널(안정/베타).
- **자동 업데이트 서명 검증**: 내려받은 exe 해시/서명 확인 후 교체(보안).
- **텔레메트리(옵트인)**: 엔진별 성공률·오류 수집으로 품질 개선.

## 🧪 품질 보증

- **단위/통합 테스트**: OCR 엔진 계약, 이미지 로딩, 업데이터 버전 비교 로직.
- **골든 이미지 회귀 테스트**: 대표 이미지들의 인식 결과 스냅샷 비교.
- **CI(빌드만)**: DevExpress 없이 컴파일 가능한 부분만 검증(엔진/업데이터 로직 분리).
